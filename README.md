# BikeDealsBot 2.0
Телеграм-бот ([@bikedealsbot](https://t.me/bikedealsbot)) для отслеживания цен в веломагазинах. Поддерживаются:
* chainreactioncycles.com
* bike-components.de
* starbike.com
* bike-discount.de

## Реализация
* Python 3.10+
* фреймворк: aiogram
* база данных: MongoDB
* запуск фоновых заданий: Advanced Python Scheduler
